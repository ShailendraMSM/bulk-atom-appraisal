# Atom Bulk Valuation Tool

A beautiful web application for bulk domain appraisal using the Atom.com API.

## Features

- ✅ Bulk domain validation
- ✅ Support for multiple TLDs (.com, .net, .org, .co, .ai, .io, .xyz, .co.uk, .app, .in)
- ✅ Real-time progress tracking
- ✅ Detailed CSV export with all appraisal data
- ✅ Beautiful responsive UI
- ✅ Error handling and reporting
- ✅ Serverless architecture (Cloudflare Pages + Functions)

## Technology Stack

- HTML5
- CSS3
- Vanilla JavaScript (ES6+)
- Cloudflare Pages (Hosting)
- Cloudflare Pages Functions (API Proxy)
